# Hash Tables - Data Structure Practice Problems

## Overview
This repository contains a collection of 10 real-world algorithmic problems solved using **Hash Tables**, **Sets**, and **Maps** in Java. These implementations focus on optimizing time complexity (specifically achieving O(1) lookups), handling high-frequency data operations, and managing concurrency.

## 🚀 Projects Implemented

1. **Social Media Username Checker** (`UsernameChecker.java`)
    - O(1) availability lookups and attempt frequency tracking.
2. **Flash Sale Inventory Manager** (`InventoryManager.java`)
    - High-traffic stock tracking with a FIFO `LinkedHashMap` waiting list.
3. **DNS Cache with TTL** (`DNSCache.java`)
    - Time-to-Live expiration logic with nanosecond performance benchmarking.
4. **Plagiarism Detection System** (`PlagiarismDetector.java`)
    - Document similarity analysis using n-gram extraction and Hashing.
5. **Real-Time Analytics Dashboard** (`AnalyticsDashboard.java`)
    - Multi-dimensional data tracking for unique visitors and traffic sources.
6. **Distributed Rate Limiter** (`APIRateLimiter.java`)
    - Token Bucket algorithm implementation for API gateway protection.
7. **Search Engine Autocomplete** (`AutocompleteSystem.java`)
    - Prefix matching and query frequency sorting.
8. **Parking Lot Management** (`ParkingLot.java`)
    - Custom Hash Table built from scratch using Open Addressing and Linear Probing.
9. **Financial Transaction Analyzer** (`TransactionAnalyzer.java`)
    - O(1) complement lookups for Two-Sum fraud detection and duplicate identification.
10. **Multi-Level Cache System** (`MultiLevelCache.java`)
    - L1 (Memory), L2 (SSD), and L3 (Database) architecture with LRU eviction and promotion logic.
